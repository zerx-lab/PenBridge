# 发布文章

## 多平台发布

PenBridge 支持一键发布文章到多个技术平台，告别繁琐的复制粘贴操作。

### 支持的平台

| 平台 | 状态 | 功能支持 |
|------|------|----------|
| 腾讯云开发者社区 | ✅ 已支持 | 发布、草稿、分类、标签、专栏 |
| 掘金 | ✅ 已支持 | 发布、草稿、分类、标签、封面图 |
| CSDN | 🚧 开发中 | - |
| 思否 | 📋 规划中 | - |
| 知乎 | 📋 规划中 | - |
| 博客园 | 📋 规划中 | - |

想要支持更多平台？[参与功能调研](https://penbridge.vercel.app/survey)

### 发布流程

1. 在编辑器中完成文章撰写
2. 点击右上角「发布」按钮
3. 选择目标平台
4. 填写平台特定信息（分类、标签等）
5. 点击「发布」或「保存草稿」

### 首次登录平台

首次发布到某个平台时，需要完成一次手动登录：

1. 点击发布按钮，选择平台
2. 弹出平台登录窗口
3. 在登录窗口中完成账号登录
4. 登录成功后，Cookie 自动保存
5. 后续发布无需重复登录

> 登录状态会定期自动检测，如过期会提醒重新登录。

## 腾讯云开发者社区

### 发布设置

| 设置项 | 说明 | 必填 |
|--------|------|------|
| 分类 | 文章所属分类 | ✅ |
| 标签 | 文章标签，最多5个 | ❌ |
| 专栏 | 关联的专栏 | ❌ |
| 摘要 | 文章摘要 | ❌ |

### 发布选项

- **立即发布**: 文章立即上线
- **保存草稿**: 保存到平台草稿箱
- **定时发布**: 设置指定时间自动发布

### 文章同步

- 支持更新已发布的文章
- 自动记录文章在平台的 ID
- 可查看发布状态和链接

## 掘金

### 发布设置

| 设置项 | 说明 | 必填 |
|--------|------|------|
| 分类 | 文章分类（前端、后端等） | ✅ |
| 标签 | 文章标签，最多选择已有标签 | ✅ |
| 封面图 | 文章封面图片 | ❌ |
| 摘要 | 文章简介 | ❌ |

### 封面图

- 支持上传本地图片作为封面
- 也可使用文章中已有的图片
- 建议尺寸: 800x450 像素

### 发布选项

- **立即发布**: 文章立即上线
- **保存草稿**: 保存到掘金草稿箱
- **定时发布**: 设置指定时间自动发布

## 图片处理

### 自动上传

发布时，文章中的图片会自动处理：

1. **本地图片**: 自动上传到目标平台图床
2. **外链图片**: 保持原链接不变
3. **平台图片**: 已在该平台的图片保持不变

### 图片格式

支持的图片格式：
- PNG
- JPEG/JPG
- GIF
- WebP

### 图片大小

- 单张图片建议不超过 5MB
- 过大的图片可能导致上传失败

## 发布状态

### 状态类型

| 状态 | 说明 |
|------|------|
| 未发布 | 文章从未发布到该平台 |
| 已发布 | 文章已成功发布 |
| 草稿 | 已保存到平台草稿箱 |
| 待更新 | 本地有修改，需要同步更新 |

### 同步管理

在文章编辑页面可以查看：

- 各平台的发布状态
- 发布时间
- 平台文章链接
- 最后同步时间

## 常见问题

### 登录状态失效

平台 Cookie 可能会过期，解决方法：

1. 进入设置 → 平台账号
2. 点击对应平台的「重新登录」
3. 完成登录后即可继续发布

### 发布失败

常见原因：
- 网络连接问题
- 登录状态过期
- 文章内容不符合平台规范
- 图片上传失败

解决方法：
1. 检查网络连接
2. 确认登录状态有效
3. 查看错误提示详情
4. 尝试重新发布

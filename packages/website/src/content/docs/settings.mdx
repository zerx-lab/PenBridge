# 设置

## 应用设置

PenBridge 提供丰富的配置选项，满足不同用户的使用需求。

## 通用设置

### 主题

- **浅色模式**: 明亮的界面主题
- **深色模式**: 护眼的暗色主题
- **跟随系统**: 自动跟随操作系统设置

### 语言

目前支持中文界面，更多语言持续添加中。

### 启动设置

- **开机自启动**: 系统启动时自动运行（仅桌面应用）
- **最小化到托盘**: 关闭窗口时最小化到系统托盘

## 编辑器设置

### 字体设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 字体 | 编辑器使用的字体 | 系统默认 |
| 字号 | 文字大小 | 16px |
| 行距 | 行间距倍数 | 1.6 |

### 编辑器行为

- **自动保存**: 编辑内容自动保存
- **保存间隔**: 自动保存的时间间隔
- **拼写检查**: 开启/关闭英文拼写检查

## 平台账号

管理已配置的发布平台账号。

### 支持的平台

| 平台 | 登录方式 | 状态检测 |
|------|----------|----------|
| 腾讯云开发者社区 | 浏览器登录 | 每日自动 |
| 掘金 | 浏览器登录 | 每日自动 |

### 账号管理

每个平台可以：

- **查看状态**: 显示登录是否有效
- **重新登录**: 登录状态失效时重新登录
- **退出登录**: 清除保存的登录状态

### 登录状态

| 状态 | 说明 |
|------|------|
| 已登录 | Cookie 有效，可正常发布 |
| 已过期 | Cookie 失效，需重新登录 |
| 未登录 | 从未登录过该平台 |

## AI 配置

配置 AI 助手使用的服务商和模型。

### 服务商管理

支持添加多个 AI 服务商：

| 服务商 | 需要 API Key | Base URL |
|--------|--------------|----------|
| OpenAI | ✅ | 可选 |
| 智谱 AI | ✅ | 内置 |
| DeepSeek | ✅ | 内置 |
| GitHub Copilot | ❌ (OAuth) | 内置 |
| 自定义 | ✅ | 必填 |

### 配置步骤

1. 点击「添加服务商」
2. 选择服务商类型
3. 填写 API Key（或完成 OAuth 授权）
4. 选择可用的模型
5. 设置是否为默认服务商

### 模型管理

每个服务商可配置多个模型：

- 添加/删除可用模型
- 设置默认使用的模型
- 查看模型能力（是否支持工具调用、推理等）

### GitHub Copilot 配置

无需 API Key，通过 OAuth 授权：

1. 点击「连接 GitHub Copilot」
2. 跳转到 GitHub 完成授权
3. 返回应用后自动连接
4. 可随时断开连接重新授权

## 邮件配置

配置邮件通知服务，接收发布结果通知。

### SMTP 配置

| 配置项 | 说明 | 示例 |
|--------|------|------|
| SMTP 服务器 | 邮件服务器地址 | smtp.qq.com |
| 端口 | 服务器端口 | 465 (SSL) / 587 (TLS) |
| 用户名 | 发件邮箱 | your@qq.com |
| 密码/授权码 | 邮箱密码或授权码 | - |
| 发件人名称 | 邮件显示的发件人 | PenBridge |

### 常用邮箱配置

**QQ 邮箱**:
- 服务器: smtp.qq.com
- 端口: 465 (SSL)
- 需使用授权码（非登录密码）

**163 邮箱**:
- 服务器: smtp.163.com
- 端口: 465 (SSL)
- 需开启 SMTP 服务

**Gmail**:
- 服务器: smtp.gmail.com
- 端口: 587 (TLS)
- 需开启两步验证并使用应用密码

### 通知设置

选择接收通知的类型：

- ✅ 定时发布成功
- ✅ 定时发布失败
- ✅ 登录状态过期
- ✅ 重试失败通知

### 测试邮件

配置完成后，点击「发送测试邮件」验证配置是否正确。

## 数据管理

### 数据存储

数据存储在本地 SQLite 数据库：

| 部署方式 | 数据位置 |
|----------|----------|
| Windows | `%APPDATA%/pen-bridge/data/` |
| macOS | `~/Library/Application Support/pen-bridge/data/` |
| Linux | `~/.config/pen-bridge/data/` |
| Docker | `/app/data/` (容器内) |

### 数据导出

支持导出所有数据：

1. 进入设置 → 数据管理
2. 点击「导出数据」
3. 选择导出格式（JSON 或 ZIP）
4. 可选择加密导出

**导出内容包括**:
- 所有文章内容
- 文件夹结构
- AI 聊天记录
- 应用配置（不含敏感信息）

### 数据导入

从备份恢复数据：

1. 点击「导入数据」
2. 选择备份文件
3. 确认导入（将覆盖现有数据）
4. 重启应用生效

### 加密备份

导出时可选择加密：

- 使用密码加密备份文件
- 导入时需输入正确密码
- 保护敏感的 API Key 等信息

## 用户管理

支持多用户管理（适合团队使用）。

### 管理员账户

首个创建的账户为管理员：

- 可创建/删除其他用户
- 可重置用户密码
- 可查看所有用户的文章

### 普通用户

普通用户权限：

- 管理自己的文章
- 使用发布功能
- 使用 AI 助手
- 修改个人设置

### 账户操作

| 操作 | 管理员 | 普通用户 |
|------|--------|----------|
| 修改密码 | ✅ | ✅ (仅自己) |
| 创建用户 | ✅ | ❌ |
| 删除用户 | ✅ | ❌ |
| 重置密码 | ✅ | ❌ |

## 配置文件

### 文件位置

配置文件存储位置：

| 系统 | 路径 |
|------|------|
| Windows | `%APPDATA%/pen-bridge/config.json` |
| macOS | `~/Library/Application Support/pen-bridge/config.json` |
| Linux | `~/.config/pen-bridge/config.json` |

### 手动编辑

高级用户可直接编辑配置文件：

```json
{
  "theme": "dark",
  "editor": {
    "fontSize": 16,
    "lineHeight": 1.6,
    "autoSave": true
  }
}
```

> 编辑配置文件前建议先备份

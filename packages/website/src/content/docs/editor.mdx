# 文章编辑

## Markdown 编辑器

PenBridge 内置基于 [Milkdown](https://milkdown.dev/) 的 Markdown 编辑器，提供沉浸式所见即所得的写作体验。

### 核心功能

- **实时预览**: 所见即所得，无需切换预览模式
- **斜杠命令**: 输入 `/` 快速插入标题、列表、代码块、表格等
- **代码高亮**: 支持多种编程语言语法高亮
- **目录导航**: 右侧自动生成文章目录，点击快速跳转
- **自动保存**: 编辑内容自动保存，无需手动操作
- **拼写检查**: 英文拼写错误实时提示

### 斜杠命令

输入 `/` 后可选择：

| 命令 | 说明 |
|------|------|
| `/heading1` | 一级标题 |
| `/heading2` | 二级标题 |
| `/heading3` | 三级标题 |
| `/bullet` | 无序列表 |
| `/ordered` | 有序列表 |
| `/task` | 任务列表 |
| `/code` | 代码块 |
| `/quote` | 引用块 |
| `/table` | 表格 |
| `/divider` | 分割线 |
| `/image` | 插入图片 |

### 快捷键

| 功能 | Windows/Linux | macOS |
|------|--------------|-------|
| 加粗 | `Ctrl+B` | `Cmd+B` |
| 斜体 | `Ctrl+I` | `Cmd+I` |
| 链接 | `Ctrl+K` | `Cmd+K` |
| 代码 | `Ctrl+\`` | `Cmd+\`` |
| 保存 | `Ctrl+S` | `Cmd+S` |
| 撤销 | `Ctrl+Z` | `Cmd+Z` |
| 重做 | `Ctrl+Shift+Z` | `Cmd+Shift+Z` |
| 搜索 | `Ctrl+F` | `Cmd+F` |

## 图片管理

### 插入图片

支持多种方式插入图片：

1. **拖拽上传** - 直接拖拽图片到编辑器
2. **粘贴上传** - 从剪贴板粘贴图片 (`Ctrl/Cmd+V`)
3. **斜杠命令** - 输入 `/image` 选择本地图片
4. **Markdown 语法** - 输入 `![alt](url)` 引用网络图片

### 图片存储

- **本地存储**: 默认存储在应用数据目录
- **平台上传**: 发布时自动上传到目标平台
- **自动清理**: 定期清理未被引用的图片

## Word 文档导入

支持导入 `.docx` 格式的 Word 文档：

1. 点击工具栏的「导入」按钮
2. 选择 Word 文档文件
3. 文档内容自动转换为 Markdown
4. 图片自动提取并保存

### 导入支持

- 标题层级
- 加粗、斜体、下划线
- 有序/无序列表
- 表格
- 嵌入图片
- 代码块

## 文件管理

### 文件树

编辑器左侧是文件树，支持：

- **创建** - 右键菜单创建文件夹或文章
- **重命名** - 双击或右键重命名
- **删除** - 右键菜单删除（删除前确认）
- **拖拽** - 拖拽移动文件到其他文件夹
- **排序** - 拖拽调整文件顺序

### 文件夹

- 支持多级嵌套文件夹
- 点击文件夹展开/收起
- 文件夹可包含子文件夹和文章

## 目录导航

编辑器右侧显示文章目录：

- 根据标题自动生成
- 显示标题层级结构
- 点击标题快速跳转
- 当前位置高亮显示

## 编辑器设置

在设置页面可自定义编辑器：

- **字体**: 选择编辑器字体
- **字号**: 调整文字大小
- **行距**: 调整行间距
- **主题**: 浅色/深色模式
